The system can be run the eclipse IDE.

1 - Import the project into eclipse
2 - Open classes from mannySandhu.neuroevolution.tce.train
3 - All training algorithms used in the project are available in this package (The parameters are already set)
4 - To run a training algorithm simply hit run
5 - Test cases can be viewed in the res folder and selectected for training in package mannySandhu.neuroevolution.tce/MlgData.java
